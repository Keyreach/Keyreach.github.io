nelistvw.js docs

Get nelistvw.js

Usage

var listview = new neListVw(options)

Options

options

Type: object

Keys:

elem
HTMLElement, where listview will be mounted
multi
Boolean, defining if it is possible to select multiple items
key
String, key in dataset, which will be used for identifying items, e.g. for .value setter
columns
Function, called with HTML element, dataset item and index passed
Object, where keys are keys in dataset, and value is either function that will be applied to dataset value with corresponding key or object, containing this function as 'filter' and applied styles as 'style'
style
Object, containing keys 'normal' and 'selected', representing CSS class used for all and selected items

Methods

.render()

Force redraw widget

.update(data)

Update dataset

.add(item)

Add item to dataset

.cut(index)

Cut item from dataset
